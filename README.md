# SirensMoon
★doxygen　Javadocスタイルによるコメントを書く。

1．各ファイルの先頭に書くコメント
```
/**
* @file ファイル名 main.cppとか
* @brief ファイルの説明
* @author 作成者名
* @date 作成日
*/
```
2.各関数の前行に書くコメント
```
 /**
 * @brief 関数の説明
 * @param [引数の名前] 引数の説明　
 * @return [型] 戻り値の説明
 */
 ```
3.変数、enumの書くコメント
```
/** 変数の説明 */
int a {0}
```
あるいは
```
int a {0};//<変数の説明
```

★命名ルール

関数名、クラス名：アッパーキャメル

変数名：ローワーキャメル


★コミットコメントルール

先頭にプレフィックスを付ける。該当するものが無ければ相談
